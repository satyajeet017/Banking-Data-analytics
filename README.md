# 🏦 Banking Data Analytics Project

## 📌 Project Overview

This project focuses on analyzing banking customer and loan data using SQL and Power BI to generate actionable business insights. The analysis helps understand customer behavior, loan performance, transaction trends, and key banking KPIs.

---

## 🎯 Objectives

- Analyze customer banking transactions.
- Track loan performance and approval trends.
- Monitor key banking KPIs.
- Identify customer segments and business opportunities.
- Create interactive dashboards for decision-making.

---

## 🛠️ Tools & Technologies

- SQL
- Power BI
- Microsoft Excel / CSV
- Data Analytics
- Data Visualization

## 🗄️ SQL Analysis

The project includes SQL queries for:

- KPI Calculation
- Customer Analysis
- Loan Analysis
- Transaction Analysis
- Business Reporting

Sample Query:

```sql
SELECT
    SUM(Credit_Amount) AS Total_Credit,
    SUM(Debit_Amount) AS Total_Debit
FROM banking_data;
```
